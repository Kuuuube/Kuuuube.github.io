Kuuube's Japanese Tools

Kana Quiz and Kana Quiz Sounds

Kana Quiz is a Hiragana, Katakana, and Hentaigana learning tool.

Most users will want only Hiragana and Katakana. Select the top Kana Quiz option for Hiragana and Katakana or the bottom 変体仮名クイズ option for Hentaigana.

Kana Quiz Sounds is a similar Hiragana and Katakana learning tool but uses audio rather than Romaji.

Minimal Pairs

A backup of the コツ minimal pairs test.

Contains extra convenience features and performance optimizations.

Kanji Punt Gun

Provides just about every static visual way possible of searching kanji. All at the same time.

This includes searching by radical, component, four corner, skip code, de roo, stroke count, composition, and partial word matches.

Crop Theft Anki Note Type

A simple and robust Anki note type.

Kanji Grid Kuuube for Anki

An up to date and maintained Kanji Grid addon. View the kanji in your Anki decks in a beautiful grid format.

Japanese Text Analyzer

Analysis tool primarily for ocr files in Mokuro processed manga. Also supports miscellaneous files.

Hentaigana Input

A simple way to type Hentaigana characters.

For more info or to download the offline desktop app see: Hentaigana Input Readme.

Yomitan Dictionaries

A collection of Yomitan dictionaries created or modified by me.

JMdict PyGlossary

Automatically generated JMDict dictionaries for use with PyGlossary supported formats. Updates weekly.

Many formats are supported but have not been added due to lack of demand. If you would like to have a format added, please ask.
