Color Key:Key for somewhat working pens/mice:
3
: Perfectly working user tested or specified by WacomIf any of the highlighted features are not listed, those features do not detect at all or the pen in question does not support these features. If there is a ? instead of a number, the tester was unable to test that feature.
4
: User tested to not workL0: Position works perfectlyP0: Pressure works PerfectlyB0: Pen/mouse buttons work perfectlyT2: Pen tip does not workE0: Eraser detection works perfectly
5
: User tested to somewhat workL1: Position has lag, noise, or precision issuesP1: Pressure is usable but levels are wrongB1: Fewer than all pen/mouse buttons workD2: The pen/mouse does not detect when draggingE1: Eraser functions the same as the pen tip
6
: Untested, may match tablets with similar compatibilityL2: Position has extreme lag, noise, or precision issuesP2: Effectively digital pressure levels (0-1)B2: Pen/mouse buttons send unusable dataH2: The pen/mouse does not detect when hoveringE2: Eraser side does not work
7
: Untested and no good assumptions can be made
8
All compatibility ratings in this sheet apply to the tablet itself. Wacom drivers almost always block pens with partial compatibility. Using OpenTabletDriver is recommended.
